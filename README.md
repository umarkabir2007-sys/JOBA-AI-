# Joba AI – CV Builder, Job Matcher & Scholarship Finder

Build professional CVs, find AI-matched jobs, and discover scholarships – all in one place.

## Features

- **CV Builder** – Create ATS-friendly CVs, pay ₦500 to download PDF.
- **AI Job Matcher** – Upload CV to see matching jobs. Pay ₦1,000 to unlock apply links.
- **Scholarship Finder** – Search scholarships worldwide. Pay ₦500 to unlock requirement analysis & official links.
- **Testimonials** – Real success stories from users.
- **Dark/Light theme** toggle.

pip install -r requirements.txt
